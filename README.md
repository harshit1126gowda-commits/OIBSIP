# BMI Calculator - Health Tracker

A Python-based BMI Calculator application developed during my Python Internship at OIBSIP. This project provides a user-friendly GUI to calculate BMI, store records, search history, and visualize BMI data.

## Features

- Calculate BMI using height and weight
- BMI category display
- Input validation
- Reset functionality
- Store records using SQLite database
- View BMI history
- Search records by name
- Delete history
- BMI graph visualization

## Technologies Used

- Python
- Tkinter (GUI)
- SQLite Database
- Matplotlib
